# [MCD VIRAL: Visual-Inertial-Ranging-Lidar Multi-Campus Dataset with Continous-Time Ground Truth](https://ntu-aris.github.io/ntu_viral_dataset/)


The dataset is collected from multi-sensor suites featuring:

* 3D spinning lidar
* Epicyle pattern lidar
* Stereo cameras
* Multiple IMUs
* UWB nodes.

Captured over large-scale areas accross the globe.
